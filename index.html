Trigger conditions — all must be true
  • Restaurant added to NeoTaste in user's city within last 14 days
  • Restaurant has 0 completed structured reviews
  • User is Foodie level or above
  • User has not completed a mission in the last 7 days
  • No one currently holds the active slot for this restaurant
Slot assignment logic
Priority order: NeoTaste Hero → Local → Cityguide → Foodie. If multiple users at the same level qualify, assign by proximity. Only 1 slot active at a time — selected user has 48h to book. Others remain queued with no notification until slot opens.
Queue fallback
If the active user does not book within 48h, the slot expires silently. No failure message is sent. The next queued user receives a fresh notification. The 7-day review window still applies to the new invitee.

Mission review — what's different from standard
  • Header: "You're the first. Make it count." — not "Glad you liked it!"
  • Star rating sits at the top — existing NeoTaste behaviour, unchanged
  • Photo upload follows — optional, existing behaviour, unchanged
  • Three structured prompts follow — required for first-mover credit
  • Critic's pick text field follows — optional, one sentence, new
  • Profile name and avatar preview closes the screen — new
  • Level progress bar more prominent — this review carries more weight toward levelling up
  • "Later" button still present — never remove the escape route
Full screen sequence — field by field
1. Star rating — existing. Required. Unchanged from standard review.

2. Photo upload — existing. Optional. Unchanged from standard review.

3. Occasion prompt — new. Required for first-mover credit. Solo lunch / Date night / Group dinner / Quick bite.

4. Vibes prompt — new. Required for first-mover credit. Cosy and quiet / Social and fun / Cool discovery / Chill / Great for dates / Local favourite.

5. Come back? prompt — new. Required for first-mover credit. Yes, without a doubt / Yes, if the deal is on / Probably not / No.

6. Critic's pick — new. Optional. One sentence, character-limited. Placeholder: "What's the one thing people need to know?" If completed, this line appears permanently on the restaurant page under her name.

7. Profile preview — new. Shows her first name and avatar as they'll appear on the restaurant page. If profile is incomplete, a nudge to fill it in. If complete, a confirmation: "This is how you'll appear as the first reviewer."
The three structured prompts
Prompt 1 — Occasion: Solo lunch / Date night / Group dinner / Quick bite

Prompt 2 — Vibes: Cosy and quiet / Social and fun / Cool discovery / Chill / Great for dates / Local favourite

Prompt 3 — Would you come back? Yes, without a doubt / Yes, if the deal is on / Probably not / No

All three must be answered for first-mover credit. Stars + photos alone, or partial prompt completion, generate data but do not qualify for the payoff.
Critic's pick — the optional one-sentence review
A single text field, character-limited (100 chars max). Placeholder copy: "What's the one thing people need to know?" — frames it as a pick, not a review obligation.

If she completes it: the sentence appears permanently on the restaurant detail page directly under her first name and avatar in the "First taste by" section.

If she skips it: the first-mover credit still applies. Her name and avatar appear on the restaurant page with no quote. The quote slot is simply absent — no empty placeholder shown to other users.
Profile name and avatar preview
Not a separate settings screen — a read-only preview at the bottom of the review screen showing exactly how she'll appear as the first reviewer.

If profile is complete: Shows her first name, avatar colour/initials, and level badge. Copy: "This is how you'll appear on this restaurant's page."

If profile is incomplete (no avatar set, display name missing): Shows a placeholder with a nudge — "Add a photo to your profile to complete your reviewer identity." Links to profile settings. She can skip this and still submit — the nudge is not a blocker.
User taps Later — grace period
Mission credit held for 7 days from redemption date. One gentle in-app re-prompt on next app open: "You still have first-reviewer status at [Restaurant]. Leave your pick before it's gone." After 7 days, credit expires. No new mission issued for this restaurant — first review credit goes unclaimed.
